# Contributing

Thanks for helping make CrabiX clearer and easier to evaluate.

This public repository is for product information, release downloads, security reporting, and community entry points. It is not the full application source repository.

## Good Contributions

- Fix broken links.
- Improve public documentation.
- Clarify download or release instructions.
- Report product copy that conflicts with [crabix.ai](https://crabix.ai).
- Suggest missing platform or installation notes.

## Before Opening A Pull Request

1. Check whether the same issue already exists.
2. Keep changes small and focused.
3. Do not add private source code, secrets, credentials, internal endpoints, or unreleased claims.
4. Keep public copy in English.
5. If changing download status, update `README.md`, `docs/downloads.md`, and `llms.txt` together.
6. If changing security claims, update `SECURITY.md` and `docs/security-local-first.md` together.

## Product Claims

CrabiX is pre-launch. Do not add:

- Fake testimonials.
- Customer counts.
- Enterprise logos.
- Revenue numbers.
- Unverified benchmarks.
- "Production-proven" claims.

## Security Issues

Do not report vulnerabilities in public issues or pull requests. Use [SECURITY.md](SECURITY.md).
